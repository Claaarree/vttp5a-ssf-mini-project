CPB

Requests are HTTP "GET" from the base Url:
vttp5a-ssf-mini-project-production.up.railway.app/api

To get a list of foods for a given search term

Example: https://vttp5a-ssf-mini-project-production.up.railway.app/api/food/list?food=salad&page_offset=0

Parameters Description
path /food/list
food Name of desired food details
page_offset Number of pages to offset the results by. "0" starts from the beginning of the results.

To get food details eaten by a particular user based on the food id

Example: https://vttp5a-ssf-mini-project-production.up.railway.app/api/food/id?user_Id=7b79604a-843d-4128-8125-cb33a6b4622b&food_id=2061496&food_Id=CUSTOMf88372e6-46b3-4a3f-84d7-fa88e0144261

Parameters Description
path /food/id
food_Id Food ID
user_Id User's ID

To get entries of a particular user

Example: https://vttp5a-ssf-mini-project-production.up.railway.app/api/entries?user_Id=7b79604a-843d-4128-8125-cb33a6b4622b&from=2024-12-01&to=2024-12-10

Parameters Description
path /entries
user_Id User ID
from Optional. Gets entries from this date formatted yyyy-mm-dd
to Optional. Gets entries from this date formatted yyyy-mm-dd
